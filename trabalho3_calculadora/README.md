# Calculadora Flutter

## Descrição
Aplicativo de calculadora simples desenvolvido em Flutter,
com interface gráfica funcional e operações matemáticas básicas.

## Estrutura de Componentes

### Display
Widget responsável por exibir a expressão digitada e o resultado da operação.

### BotaoCalculadora
Widget reutilizável para os botões da calculadora.
Aceita texto, cor e ação como parâmetros.

### CalculadoraPage
Página principal que gerencia o estado da calculadora
e organiza os componentes em linhas e colunas.

## Funcionalidades
- Adição, subtração, multiplicação e divisão
- Exibição da expressão e resultado em tempo real
- Limpeza da operação (botão C)
- Apagar último caractere (botão ⌫)
- Suporte a números decimais

## Como executar

1. Certifique-se de ter o Flutter SDK instalado
2. Clone o repositório:
   git clone : https://github.com/GustavoMS1304/TrabalhoGoku.git
3. Acesse a pasta do projeto:
   cd trabalho3_calculadora
4. Instale as dependências:
   flutter pub get
5. Execute o projeto:
   flutter run
