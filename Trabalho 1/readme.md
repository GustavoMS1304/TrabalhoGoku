## Objetivo
Configurar o ambiente de desenvolvimento Dart e executar um programa simples que imprime "Hello World" no terminal.

## Como executar

1. Certifique-se de ter o Dart SDK instalado
2. Clone o repositório: https://github.com/GustavoMS1304/TrabalhoGoku.git
3. Acesse a pasta do projeto
4. Execute o comando: dart run main.dart
